The Romans, as you know, used latin letters to write the numerals. It's believed that the Roman numeral system is a classic example of non-positional numeral system, in which the value (which denotes the numeral) doesn't depend on the position in the number. Here's numbers and corresponding letters:

1=I
5=V
10=X
50=L
100=C
500=D
1000=M

For example, the number 3 in the Roman numerial system will be denoted as III. However, not everything is so simple, and it's not a completely non-positional numerial system, because in the Roman numeral system there is an additional rule that affects the amount that denotes a number, depending on its position. This rule prohibits the use of the same numbers more than 3 times in a row, so 3 is III, and 4 is IV, and I(1) is in front of V(5), denotes subtraction, so that 1 is actually equal to -1.
